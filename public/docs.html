POST/auth/login
Public

Login-or-Register: if the user exists, login; if not, auto-register then return JWT.

Request body:
{
  "phoneNumber": "13xxxxxxx",
  "password": "PlainText123",
  "name": "Optional Name"
}
Responses:
  • 201: { "token": "<jwt>", "created": true } (new user)
  • 200: { "token": "<jwt>", "created": false } (existing user)
  • 400: missing fields
  • 401: invalid credentials (existing user, wrong password)
  • 500: internal error

Notes

  • Base URL: http://localhost:3000
  • Content-Type: application/json
  • Use header: Authorization: Bearer <jwt> for protected APIs.
  • Passwords hashed with bcrypt (10 rounds).
